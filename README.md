# Station 93

Game based on the tabletop RPG Traveller. Made in C++20. This project uses [BearLibTerminal](https://github.com/cfyzium/bearlibterminal) for graphics.

## How to build  

1. Make sure you are in the Developer Powershell for VS

2. Run build.ps1

3. Usage:

`-Mode release` for building in release mode.

`-Mode debug` for building in debug mode.
